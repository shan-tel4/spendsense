# SpendSense

SpendSense is a Python command line application for tracking personal income and expenses.  
It allows users to record transactions, analyse spending patterns, generate insights, and export monthly reports.

## Features
- Add income and expense transactions
- Persistent CSV storage
- View recent transactions
- Calculate total income, expenses, and balance
- Generate monthly reports
- Expense category breakdowns
- Spending insights such as averages and largest expense
- Export monthly reports to CSV

## Tech Stack
- Python
- CSV file storage
- Python standard library only
